Audience Test Results

During the user testing session I preented my working prototype to two users I hadn't met before. Each round began with 5 minutes of the user navigating my site with no talking or explanation from me. This was really helpful to witness because it allwed me to see what others pick up on that I hadn't. The user testing was beneficial because the two people who used my prototype picked up on different improvements that could be made. Here are the major takeaways:

  • Slow down the speed of the fingerspelling game
  • Show a progress visual that lets users know where they are in the site
  • Position the canvas in the first module so the user doesn't have to move their hand
  • Have an objective or goal for each game so that users don't just keep playing forever
  • Give the games more fun and playful names
