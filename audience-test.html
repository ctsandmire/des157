Audience Test Objectives

Hi! Thank you so much for participating in the testing of my project prototype. The purpose of this project is to help people learn the basics of American Sign Language.

These are the objectives: [write out on paper]

  • Navigate to each module in the desired order
  • Understand how the module works
  • Test each module and get as far as possible, regardless of skill level

Post-user test questions:

  • Can you explain what you believe the purpose of this site is?
  • Do you know ASL or any other languages? If so, how did you learn them?
  • Who do you think would use this site?
  • Walk me through what parts were confusing or unclear?
  • Is there anything that you feel is missing from this site?
